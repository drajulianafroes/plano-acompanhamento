# Plano de acompanhamento

Aplicativo estatico para calcular planos de acompanhamento com tirzepatida, despesas e resultado liquido.

## Arquivos

- `index.html`: calculadora principal
- `app.js`: logica da calculadora
- `styles.css`: estilos
- `contrato.html`: termo de acompanhamento e consentimento informado

## Publicacao no GitHub Pages

Publique a partir da branch `main`, usando a pasta raiz (`/`) como origem.
